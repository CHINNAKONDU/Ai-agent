# REAL TIME WEATHER APP

A Python Flask application for checking real-time weather conditions for villages, mandals, and districts.

## Features
- Dark Mode Background
- Village/Mandal/District Level Weather Search
- Real-time Weather Data from Open-Meteo API
- 5-Day Weather Forecast
- SQLite Database for Location Storage
- Safety Notes Display
